# Examples

This directory stores real example JSON outputs generated from the local backend.

Included examples:

- `alpha-ranked.json`
- `alpha-trends.json`
- `finance-recommend.json`
- `activity-scored.json`
- `copilot-summary.json`

Regenerate examples:

```bash
python scripts/generate_examples.py
```

The script uses the configured local API base URL from `config.json`.
